# 🔋 Smart Home Energy Consumption Prediction

## 📌 Overview
This project focuses on predicting smart home energy consumption using machine learning. It analyzes environmental and usage parameters such as temperature, humidity, time, and appliance usage to estimate future energy demand.

---

## 🎯 Objective
To develop a predictive system that helps optimize energy usage in smart homes, reduce electricity costs, and improve energy efficiency.

---

## ⚙️ Features
- Predicts energy consumption based on input parameters
- Uses machine learning models (Random Forest / Decision Tree)
- REST API built using FastAPI
- Can be integrated with Android (Kotlin) applications
- Real-time prediction support

---

## 🧠 Technologies Used
- Python
- Scikit-learn
- Pandas & NumPy
- FastAPI
- Joblib
- Google Colab

---

## 📊 Input Parameters
- Temperature (°C)
- Humidity (%)
- Hour (0–23)
- Number of Appliances

---

## 📈 Output
- Predicted Energy Consumption
